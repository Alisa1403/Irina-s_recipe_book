Салат на зиму из баклажан


Ингредиенты:

по 10 штук:

баклажан

помидор

сладкого перца

моркови

лука

рубленная зелень (у меня петрушка, укроп, кинза, сельдерей по 1/2 пучка каждой)


Этапы приготовления

  1. Баклажаны режем кружками, остальные овощи - кубиками, морковь трем на терке.

  2. Баклажаны посолить, через час отжать.

  3. Все овощи обжарить по отдельности, затем смешать. Всыпать рубленную зелень. По

  4. По вкусу посолить, поперчить, влить 3 ст.л. 9% уксуса.
    Я еще добавила мелко порезанный чеснок (примерно полголовки) и острый перчик (совсем чуть-чуть, несколько кружочков).

  5. Разложить по пол литровым банкам.
    Стерилизовать час.

  6. Из указанного количества у меня получилось 8 вот таких баночек (по 520 гр.)
